# Dossiers des books photo

Créez un dossier par série et ajoutez vos fichiers :

| Dossier | Série | Appareil |
|---------|--------|----------|
| `nantes-s100fs/` | Street Nantes | Fujifilm FinePix S100FS |
| `stralsund-s100fs/` | Fête médiévale Stralsund | Fujifilm FinePix S100FS |
| `autoportraits-r50/` | Autoportraits | Canon EOS R50 |
| `angers-street-r50/` | Street Angers | Canon EOS R50 |
| `zoo-doue-r50/` | Zoo Doué-la-Fontaine | Canon EOS R50 |

Dans chaque dossier :

- `cover.jpg` (ou `cover.JPG`) — couverture
- `01.jpg`, `02.jpg`, … — pages du book

Les chemins sont déclarés dans `src/js/book-images.js` (chargement automatique sur `projets.html`).
Formats web uniquement (JPG/PNG) — pas de fichiers RAW (.CR3, .ARW, etc.) dans le manifeste.

Le book **manifestations-angers-2025** pointe vers Instagram [@louka.cr3](https://www.instagram.com/louka.cr3/) (fichiers originaux non conservés).
